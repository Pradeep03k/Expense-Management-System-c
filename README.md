# Expense Management System (C)

## Overview

Expense Management System is a console-based application developed in C that helps users manage their daily expenses and monthly budget. The system allows users to add, view, update, and search expense records, calculate total expenses, and track the remaining budget based on monthly income.

## Features

* Add new expenses
* View all expenses
* Update existing expenses
* Search expenses by ID
* Calculate total expenses
* Track monthly income
* Display remaining budget
* Simple menu-driven interface
